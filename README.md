# Student Academic Performance Analysis

This repository contains a data science project developed using Jupyter Notebook.
It focuses on analyzing student academic performance using data analysis techniques.

## Objectives
- Analyze student academic data
- Understand performance trends
- Visualize results using graphs

## Tools & Technologies
- Python
- Jupyter Notebook (Anaconda)
- Pandas
- NumPy
- Matplotlib
- Scipy

## Project Files
- Jupyter Notebook file (.ipynb)
- Dataset file (if used)

## How to Run
1. Open Anaconda Navigator
2. Launch Jupyter Notebook
3. Open the `.ipynb` file
4. Run all cells
